# Array methods

## `push()`, `pop()`:

| Syntax            | Description                                        |
| ----------------- | -------------------------------------------------- |
| `arr.push(value)` | Thêm phần tử `value` vào cuối arr                  |
| `arr.pop()`       | Xóa phần tử cuối cùng của arr và trả về phần tử đó |

```js
const arr = [1, 2, 3, 4, 5];
arr.push(6); // [1,2,3,4,5,6]
arr.pop(); // [1,2,3,4,5]
```
