# Giới thiệu

- Array là mảng gồm các phần tử trong nó (các phần tử có thể có kiểu dữ liệu khác nhau).
- Array là một Mutable Object (nghĩa là có thể truy cập từng phần tử thông qua chỉ số và sửa đổi nó một cách trực tiếp).
- Ví dụ:

```js
const arr = [1, 2, 3, 4, 5];
console.log(arr[0]);
arr[0] = 10;
console.log(arr[0]);
//Output: 1 10
```
