# Toán tử số học, bitwise

| Toán tử | Ý nghĩa       |
| ------- | ------------- |
| `+`     | Cộng          |
| `-`     | Trừ           |
| `*`     | Nhân          |
| `/`     | Chia          |
| `**`    | Mũ            |
| `%`     | Chia dư       |
| `&`     | AND           |
| \|      | OR            |
| `~`     | NOT           |
| `^`     | XOR           |
| `<<`    | Dịch trái     |
| `>>`    | Dịch phải     |
| `++`    | Tăng 1 đơn vị |
| `--`    | Giảm 1 đơn vị |
